# Yuxi Flash Studio for macOS

中文 | [English](#english)

---

## 中文

Yuxi Flash Studio 是一个面向 macOS 的安卓刷机工具箱，集成 **ADB / Fastboot / Qualcomm EDL(9008)** 常用流程，目标是开箱即用。

### 功能特性

- ADB 设备状态刷新与重启快捷操作
- Fastboot 设备检测与基础控制
- Qualcomm EDL(9008) 设备探测
- macOS 风格液态玻璃界面
- 统一日志面板与中文交互文案

### 运行环境

- macOS 15+
- Swift 6.2+

### 本地运行

```bash
swift build
swift run AndroidToolbox
```

### 测试

```bash
swift test
```

### 开源协议

本项目使用 **Apache-2.0** 协议，详见 `LICENSE`。

## English

Yuxi Flash Studio is a macOS Android flashing toolbox that integrates common **ADB / Fastboot / Qualcomm EDL (9008)** workflows with an out-of-the-box experience.

### Features

- ADB device refresh and quick reboot actions
- Fastboot device detection and basic controls
- Qualcomm EDL (9008) device probing
- Liquid-glass styled macOS UI
- Unified log panel with Chinese-localized UX text

### Requirements

- macOS 15+
- Swift 6.2+

### Run locally

```bash
swift build
swift run AndroidToolbox
```

### Tests

```bash
swift test
```
